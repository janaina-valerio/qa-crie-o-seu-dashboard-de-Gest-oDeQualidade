# Como Contribuir para o QA Dashboard

Agradecemos pelo interesse em contribuir com o **QA Dashboard - Gestão de Qualidade (Degust PDV Delphi)**!

## Código de Conduta

- Seja respeitoso e inclusivo
- Mantenha um tom profissional
- Foque em melhorar a experiência do usuário final (QA e Gestores)

## Como Contribuir

### 1. Reportar um Bug ou Sugerir uma Melhoria

1. Verifique se o problema já foi reportado
2. Abra uma **Issue** com o template adequado
3. Descreva o problema com clareza (passos para reproduzir, comportamento esperado, etc.)

### 2. Enviar uma Pull Request (PR)

1. Faça um **Fork** do projeto
2. Crie uma branch com um nome descritivo:
   ```bash
   git checkout -b feat/nova-funcionalidade
   # ou
   git checkout -b fix/correcao-bug
```
